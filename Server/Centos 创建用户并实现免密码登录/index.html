####ssh免密码登录

1.修改配置文件
sudo vim /etc/ssh/sshd_config

2.重启ssh服务
service sshd restart

3.生成密钥

4.导入本机
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys

5.下载到客户端并设置

6.改权限
如果希望ssh公钥生效需满足至少下面两个条件：　　　　　
1) .ssh目录的权限必须是700
2) .ssh/authorized_keys文件权限必须是600
