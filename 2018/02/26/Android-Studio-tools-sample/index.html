准备使用tools

Android Studio

注意要使用Android Studio 3.0 + 的版本

命名空间

xmlns:tools="http://schemas.android.com/tools"

然后就可以使用tools的相关属性了

在google文档中我们可以看到有很多其他可用的,具体使用效果请自行查阅文档.

如:
