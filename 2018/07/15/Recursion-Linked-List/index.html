前言

很多的面试中,都有要求实现一个单链表,这里就搞一个试试.

实现

Node.kt

//创建一个Node
class Node(val value:Int){
    var next : Node? = null
}
  1. 这个Node有一个value,它是一个必传且不可修改的参数,就直接放在了构造了
  2. next指向了下一个Node,这样就可以完成一个单向的链表,让我想起了 Message的next
  3. next默认是null,这是因为单链表最后一个next中就是null.

ListCreated.kt

class ListCreated {

    fun createList(list: List<Int>): Node? {
        //1.
        if (list.isEmpty()) {
            return null
        }
        //2.
        val currentNode = Node(list[0])
        println("before ${currentNode.value}")
        //3.
        val nextNode = createList(list.subList(1, list.size))
        if (nextNode != null) {
            println("after ${currentNode.value}")
        }
        //4.
        currentNode.next = nextNode
        return currentNode
    }
}

class ListCreated {

    fun createList(list: List<Int>): Node? {
        //1.
        if (list.isEmpty()) {
            return null
        }

        //2.
        val currentNode = Node(list[0])
        println("before ${currentNode.value}")

        //3.
        val nextNode = createList(list.subList(1, list.size))
        if (nextNode != null) {
            println("after ${currentNode.value}")
        }

        //4.
        currentNode.next = nextNode
        return currentNode
    }
}
  1. 我们需要去判断List,如果为空就直接返回null了
  2. 这里就是当前的这个Node,保存了当前的value,但是next还是为null的
  3. 这里就是递归调用的地方,传入的参数是一个subList,当然是去掉了第一个,因为第一个已经是currentNode了
  4. 把currentNode的next指向了nextNode,构成单向

MainTest.java

public class MainClass {

    public static void main(String[] args){
        List<Integer> integerList = Arrays.asList(1,4,6,2);
        ListCreated created = new ListCreated();
        Node node = created.createList(integerList);
        printAll(node);
    }

    public static void printAll(Node node){
        while (node != null){
            System.out.print(node.getValue());
            System.out.print(" ");
            node = node.getNext();
        }
    }
}

打印结果:

before 1
before 4
before 6
before 2
after 6
after 4
after 1
1 4 6 2

我们可以清楚的知道before 1,4,6,2 而没有打印,说明压入了栈,指令没有返回

after本来应该是 2,6,4,1的,为什么2没有打印出来,是因为

if (nextNode != null) {
     println("after ${currentNode.value}")
}

链表最后一个Node的next是为null的,所以没有打印
