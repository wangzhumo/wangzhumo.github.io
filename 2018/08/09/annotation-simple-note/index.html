Android中使用注解笔记

2018-08-09
Java
Java

前言

@Override 就是一个很常见的注解.

  1. 检查是否正确的复写父类方法
  2. 标志是重写方法

元注解

元注解是,额,嗯 注解的注解

  • @Retention：注解生命周期
  • @Target：作用范围
  • @Inherited：标明所修饰的注解类型是被继承的,子类会继承这个注解
  • @Documented：javadoc

这些都是由Java语言本身提供的最基础的注解,他们负责注解其他的注解,具体的作用见上文
